# Shogun

Shogun is a campaign intelligence workspace for planning, reviewing, approving, and adapting personalized cross-channel campaigns. The frontend includes a complete NestWise fintech demo with three personas, three channels, policy-readiness review, synthetic pre-flight scoring, and a Wave 2 recommendation loop.

## Run locally

```bash
npm install
npm run dev
```

The app uses a local replay experience by default and is ready to connect to the documented FastAPI endpoints through `VITE_API_BASE_URL`.

## Commands

- `npm run dev` starts the Vite development server.
- `npm run build` type-checks and builds the production bundle.
- `npm run preview` previews the production build.
